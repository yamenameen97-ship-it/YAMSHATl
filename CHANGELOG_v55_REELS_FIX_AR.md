## 🐞 المشاكل التي تم إصلاحها

### 1️⃣ ترتيب شريط التابات (التصفية) — مقلوب
**المشكلة:** التابات كانت تظهر بترتيب معكوس في الواجهة الفعلية:  
`ريلز [يسار] | متابعة [وسط] | اكتشف [يمين]` ← ترتيب LTR خاطئ

**المطلوب:**  
`اكتشف [يسار] | متابعة [وسط] | ريلز [يمين]` ← ترتيب RTL طبيعي (مطابق للصورة الأولى)

**الإصلاح:** في `frontend/src/pages/Reels.jsx`
```css
.ym-reels-tabs {
  flex-direction: row;  /* v55: كان row-reverse — تم تغييره ليعطي ترتيب RTL طبيعي */
}
```

---

### 2️⃣ زر المتابعة (+) فوق شعار Y الجانبي
**المشكلة:** علامة (+) الصغيرة كانت تظهر في **الزاوية السفلية اليسرى** للمربع البنفسجي  
**المطلوب:** يجب أن تظهر في **الزاوية السفلية اليمنى** كما في التصميم المرجعي.

**الإصلاح:**
```css
.ym-action-add .ym-action-plus {
  bottom: -7px;
  inset-inline-start: -7px;  /* v55: كان inset-inline-end — انعكس في RTL إلى اليسار */
}
```

---

### 3️⃣ زر "ريلز جديد" في الشريط السفلي
**المشكلة:**
- الزر كان صغيراً (28px) ويبدو متواضعاً
- النص "ريل جديد" ظهر مقطوعاً كـ "..."
- لا يبرز كزر إنشاء رئيسي

**الإصلاح:** في `frontend/src/components/mobile/BottomNav.jsx`
- ✅ تكبير الزر من 28px إلى **40px** على الشاشات العادية
- ✅ إضافة gradient بنفسجي + ظل بنفسجي لإبراز الزر
- ✅ تغيير النص من "ريل جديد" إلى **"ريلز جديد"** (مطابقة للصورة المرجعية)
- ✅ توسيع `max-width` للّيبل من 76px إلى **100px** حتى يظهر النص كاملاً
- ✅ إعطاء `flex: 1.2` للزر المركزي ليأخذ مساحة أكبر من باقي الأزرار
- ✅ تحديث جميع breakpoints (320/360/393/400px) لتحافظ على حجم بارز للزر

```css
.ym-nav-plus-btn {
  width: 40px;  /* v55: كان 28px */
  height: 40px;
  background-image: linear-gradient(180deg, #A78BFA 0%, #8B5CF6 55%, #7C3AED 100%);
  border-radius: 12px;
  box-shadow: 0 6px 16px rgba(139,92,246,.45);  /* ظل بنفسجي */
}
.ym-nav-label-center {
  max-width: 100px;  /* v55: كان 76px — لمنع قطع النص */

}
```

### 4️⃣ إزالة كلمة "الريلز" من أعلى الصفحة + رفع التابات
**المشكلة:** كلمة "الريلز" كعنوان أعلى التابات كانت تأخذ مساحة عمودية إضافية بدون داعٍ، وتُنزل التابات إلى أسفل الشاشة.

**المطلوب (من الصورة الأخيرة بالسهم):**
- ❌ حذف كلمة "الريلز" تماماً
- ⬆️ ترفع التابات (ريلز/متابعة/اكتشف) إلى مكان كلمة "الريلز" السابق
- ⬆️ ترفع زر الثلاث نقاط (المزيد) إلى مكان كلمة "الريلز" السابق
- ⬆️ يرفع زر كتم الصوت أيضاً ليكون بنفس المستوى

**الإصلاح:**
1. **حذف عنصر العنوان** من JSX:
   ```jsx
   {/* تم حذف <div className="ym-reels-title">الريلز</div> */}
   ```
2. **تقليل الـ padding العلوي للحاوية**: من `56px 18px 10px` إلى **`18px 18px 10px`**
3. **رفع زر كتم الصوت**: من `top: 140px` إلى `top: calc(env(safe-area-inset-top) + 18px)`
4. **رفع زر الثلاث نقاط**: من `top: 140px` إلى `top: calc(env(safe-area-inset-top) + 18px)`
5. **حذف نمط `.ym-reels-title`** بالكامل

# تقرير إصلاحات v12 — 11 يونيو 2026

## المشاكل المُصلحة

### 1. هروب منطقة الكتابة + تثبيت الهيدر السفلي
**المشكلة:** منطقة الكتابة تختفي أو تُغطّى بشريط النظام / لوحة المفاتيح على الموبايل.

**الحل:**
- `src/styles/chat-layout-fix.css`: تحديث `.yam-chat-input-wrap` ليكون
  `position: sticky; bottom: 0; z-index: 50` مع
  `padding-bottom: calc(12px + env(safe-area-inset-bottom) + var(--yam-keyboard-offset))`
  وإضافة `transform: translateZ(0)` لتفعيل GPU layer ومنع الاختفاء.
- على الموبايل (`max-width: 768px`): رفع `z-index: 60`، زيادة الـ shadow،
  وضبط `scroll-padding-bottom: 170px` للمنطقة القابلة للتمرير حتى لا تُغطّى
  آخر الرسائل بالـ composer.
- `src/pages/Chat.jsx`: مزامنة الستايل الـ inline مع نفس القيم (specificity أعلى).
- `--yam-keyboard-offset` يُحسب تلقائياً عبر `useViewportHeight` من `visualViewport`
  لإبقاء الـ composer فوق لوحة المفاتيح.

### 2. مربع النص فوق الأزرار
**المشكلة:** الـ textarea وأزرار الإيموجي/المرفق/المايك في صف واحد، والمستخدم
يريد الـ textarea في صف مستقل **فوق** الأزرار.

**الحل:** في `src/components/chat/ChatInput.jsx`:
- إعادة هيكلة الـ JSX إلى صفّين:
  - `.yam-composer-textrow` يحتوي `<textarea>` فقط (عرض كامل).
  - `.yam-composer-row` يحتوي الأزرار (😊 📎 🎤) + زر الإرسال.
- إضافة CSS مطابق لكل من الصفّين، مع تعديلات الـ media query على الموبايل
  حتى يبقى الـ textarea عريضاً والأزرار في الأسفل.

### 3. خيار تثبيت التطبيق في كل المتصفحات
**المشكلة:** بانر التثبيت كان يعتمد فقط على حدث `beforeinstallprompt` المتاح
في Chrome/Edge/Samsung فقط، فلا يظهر في Safari/iOS أو Firefox.

**الحل:** إعادة كتابة `src/components/feedback/InstallPrompt.jsx`:
- اكتشاف نوع المتصفح والنظام (`detectBrowser()`).
- إذا الحدث الأصلي متاح → استخدامه (سلوك سابق محفوظ).
- إذا لم يكن متاحاً (iOS Safari, Firefox, إلخ) → عرض **fallback banner**
  بزر "كيفية التثبيت" يفتح **modal** بتعليمات يدوية مخصصة:
  - **iOS / iPadOS:** Share → Add to Home Screen.
  - **macOS Safari:** File → Add to Dock.
  - **Firefox:** قائمة → تثبيت الموقع.
  - **Android (متصفحات أخرى):** قائمة ⋮ → إضافة إلى الشاشة الرئيسية.
  - **سطح المكتب:** قائمة المتصفح → تثبيت يام شات.
- إخفاء البانر تلقائياً إذا التطبيق مثبت بالفعل
  (`display-mode: standalone` أو `navigator.standalone`).
- زر "لاحقاً" يحفظ القرار في `localStorage` لمدة 24 ساعة.

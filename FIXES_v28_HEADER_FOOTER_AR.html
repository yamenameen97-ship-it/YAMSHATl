تقرير إصلاحات v28 YAMSHAT

هذا التقرير يلخّص التعديلات المُنفّذة على المشروع بناءً على طلباتك، مع الإشارة إلى الملفات المعنية.

1) تثبيت الهيدر العلوي والسفلي للشات

  • ✓ أُضيفت قواعد transform: none !important; filter: none !important; perspective: none !important; على .app-shell.yamshat-unified وأبنائه لمنع كسر position: fixed.
  • ✓ أُلغي will-change/transform من .ym-topbar و.ym-bottomnav.
  • ✓ صفحة Inbox أصبحت تستخدم <MainLayout> الموحّد (دون hideNav) فلا تتحرّك الأشرطة مع تمرير الصفحة.

الملفات: components/layout/MainLayout.jsx, components/mobile/MobileTopBar.jsx, components/mobile/BottomNav.jsx, pages/Inbox.jsx

2) تصغير "YAMSHAT" وإزاحتها لليمين + أزرار الستوري والمجموعات

  • ✓ تم تصغير الكلمة من 1.15rem → 0.78rem وتقليل المسافة بين الحروف.
  • ✓ أصبح الشعار في مجموعة .ym-topbar-right-cluster بأقصى اليمين بجوار زر القائمة (الترتيب الطبيعي في RTL).
  • ✓ أُضيف زرّان جديدان بمنتصف الهيدر العلوي يظهران في كل الصفحات:
    • زر ستوري → /stories
    • زر مجموعات → /groups

الملف: components/mobile/MobileTopBar.jsx

3) حذف زر إنشاء الستوري من خلفية صفحة الشات

  • ✓ أُزيل شريط .yam-story-rail بما فيه زر "إنشاء قصة" + قائمة القصص من واجهة Inbox.jsx لأنه لا يخصّها.
  • ✓ يمكن إنشاء الستوري من الهيدر العلوي (زر "ستوري") أو من شريط التنقل الموحّد.

الملف: pages/Inbox.jsx

4) زر إنشاء (+) ذكي حسب الصفحة الحالية

الصفحة الحاليةإجراء زر "+"الآلية
الرئيسية /إنشاء منشورحدث yamshat:open-composer
المجموعات /groupsإنشاء مجموعةتوجيه إلى /groups/create
الدردشة /inbox أو /chatدردشة جديدةحدث yamshat:open-new-chat → يفتح NewChatDialog
الريلز /reelsريل جديدحدث yamshat:open-reel-composer
الستوري /storiesستوري جديدحدث yamshat:open-story-composer

الملف: components/mobile/BottomNav.jsx

5) مودال "دردشة جديدة" الكامل

  • ✓ قسم "سبق التحدث معهم" يعرض آخر المحادثات من getChatThreads().
  • ✓ زر "من جهات الاتصال" يستخدم navigator.contacts.select() (Contact Picker API — Chrome Android). يفشل بأمان مع رسالة واضحة على المتصفحات غير المدعومة.
  • ✓ حقل بحث لحظي عن مشترك بالاسم عبر getUsers({ q, limit: 20 }).
  • ✓ الضغط على أي نتيجة ينتقل مباشرةً إلى /chat/:username.
  • ✓ مودال عام مُسجّل في App.jsx ويستجيب لحدث yamshat:open-new-chat من أي صفحة.

الملفات الجديدة: components/chat/NewChatDialog.jsx, التعديل في: App.jsx

ملاحظات تقنية

  • كل الواجهات الجديدة dir="rtl" وتستخدم خط Noto Sans Arabic (بديل: Tajawal).
  • كل عناصر position: fixed محميّة من تأثير transforms الأبوية.
  • التحديثات متوافقة مع الموبايل والديسكتوب (responsive).
  • ⚠ ميزة "جهات الاتصال" تعمل فقط على المتصفحات الداعمة لـ Contact Picker API (Chrome على Android بشكل رئيسي).

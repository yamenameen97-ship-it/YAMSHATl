# v87.9 — تحسين شامل للأزرار والقوائم

## الملخص
تحسين شكل وتنظيم كل الأزرار والقوائم في جميع صفحات التطبيق لتظهر بشكل مثالي، راقٍ، واحترافي — بدون تغيير أي منطق JS أو حذف أي مكون.

## الملفات المتأثرة
- **جديد:** `frontend/src/styles/yamshat-fixes-v87.9-BUTTONS-MENUS-POLISH.css`
- **معدّل:** `frontend/src/main.jsx` — إضافة استيراد الملف الجديد كآخر ملف CSS (يفوز في الـ cascade) + تحديث `BUILD_ID`.

## أبرز التحسينات

### الأزرار (`.btn`, `button`, `[role="button"]`)
- زوايا موحّدة (14px) وظلال متعددة الطبقات
- تدرج لوني احترافي لأنواع primary / success / danger / warning
- تفاعل hover مع رفع خفيف (translateY -1px) وظل أعمق
- تفاعل active مع ارتداد لطيف (scale 0.97)
- `focus-visible` بحلقة زرقاء واضحة لسهولة الوصول
- حالة معطلة أنيقة (تشبع أقل + مؤشر not-allowed)
- أحجام موحدة sm / md / lg
- أزرار أيقونية دائرية 40×40 + FAB بظل ملوّن

### القوائم المنسدلة (`dropdown-menu`, `[role="menu"]`)
- زوايا 16px + ظل نافذة عائمة + backdrop-blur
- عناصر القائمة `[role="menuitem"]` بمسافات مريحة (10×14) وزوايا 10px
- hover بلون primary شفاف + focus ring داخلي
- دعم عنصر خطر `.is-danger` بلون أحمر
- فواصل ورؤوس أقسام أنيقة
- انيميشن ظهور ناعم (fade + slide 6px)

### الشريط السفلي (BottomNav)
- خلفية شفافة مع blur احترافي
- مؤشر نشاط علوي (شريط أزرق قصير) مع أنيميشن ظهور
- ارتفاع طفيف للأيقونة النشطة
- شارات (badges) بنبض خفيف

### التبويبات (Tabs)
- خلفية موحّدة رمادية شفافة + التبويب النشط بلون سطح واضح مع ظل
- تمرير أفقي دون شريط تمرير مرئي

### Chips / Pills
- شكل بيضاوي متكامل، حالة نشطة بلون primary مع ظل ملوّن

### الشيت السفلي (BottomSheet)
- زوايا علوية 22px + مقبض سحب موحّد

### دعم RTL
- خط عربي مفضّل: Noto Sans Arabic → Tajawal → Cairo (كما يفضّل المستخدم)
- محاذاة right + تعديل مسافات الأيقونات داخل عناصر القائمة
- تبديل ذكي لاتجاه أزرار الفوتر في النوافذ المنبثقة

### إمكانية الوصول
- احترام `prefers-reduced-motion` (إلغاء الحركة كليًا)
- دعم `prefers-color-scheme: dark` تلقائيًا (ظلال أعمق، أسطح داكنة)

### حماية المكونات الحساسة
- استثناءات لفقاعات الشات، والتفاعلات (reactions/emoji)، ومشغل الرسائل الصوتية
- سمة `data-no-v879` لتعطيل التحسينات على أي عنصر خاص عند الحاجة

## طريقة التفعيل
لا حاجة لأي خطوة — ما إن يقوم Vite بإعادة البناء، سيظهر التحسين في كل الصفحات تلقائيًا. الملف الجديد يُستورد آخر شيء في `main.jsx` وبالتالي يفوز على أي أنماط سابقة.

## اختبار سريع بعد النشر
1. صفحة تسجيل الدخول والتسجيل — أزرار primary بتدرج
2. الشات — الأزرار الأيقونية دائرية، القائمة السياقية على الرسائل
3. Feed — أزرار الإعجاب/التعليق/المشاركة بشكل موحد
4. Settings — مجموعات الأزرار وحقول النموذج
5. BottomNav — مؤشر التبويب النشط
6. الملف الشخصي — التبويبات في الأعلى

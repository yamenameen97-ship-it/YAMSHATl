# 🎨 إعادة تصميم تجربة الموبايل — يمشات (Yamshat)

تمّ تحديث صفحة الموبايل لتطابق التصميم المرجعي المقدّم: واجهة داكنة أنيقة، لمسات بنفسجية، بطاقات منظّمة، وشريط سفلي حديث مع زرّ + مركزي بارز.

---

## ✨ ما الذي تغيّر؟

### 1) ملفات جديدة

| الملف | الدور |
|---|---|
| `src/styles/mobile-yamshat-redesign.css` | كل أنماط التصميم الجديد للموبايل (يُحمَّل آخر شيء ليفوز في الـ cascade) |
| `src/components/mobile/MobileTopBar.jsx` | الشريط العلوي: قائمة + لوغو YAMSHAT في الوسط + جرس إشعارات |
| `src/components/mobile/MobileTabs.jsx` | تبويبات: الرئيسية / الرائج / المتابعون مع خط سفلي بنفسجي |
| `src/components/mobile/MobileComposer.jsx` | صندوق "بماذا تفكر؟" + أيقونات صورة / GIF / إيموجي |
| `src/components/mobile/MobileFilterPills.jsx` | الكل / التحديثات / الإعلانات / المجتمع — حبّات قابلة للتمرير |
| `src/components/mobile/MobilePostCard.jsx` | بطاقة منشور بتصميم احترافي + أيقونات SVG + هاشتاجات بنفسجية |
| `src/pages/FeedMobile.jsx` | الصفحة الرئيسية للموبايل تستخدم `useSmartFeed` نفسه |

### 2) ملفات مُحدَّثة

| الملف | التعديل |
|---|---|
| `src/components/mobile/BottomNav.jsx` | إعادة بناء كامل بأيقونات SVG حديثة + زرّ + مركزي بنفسجي بارز |
| `src/layouts/MobileLayout.jsx` | ضمّ TopBar + Tabs + Content + BottomNav بهيكلية نظيفة |
| `src/pages/FeedEnhanced.jsx` | فحص `useIsMobile()` ⇒ عرض `FeedMobile` للموبايل (الديسكتوب لم يتغيّر) |
| `src/main.jsx` | تحميل ملف الأنماط الجديد كآخر استيراد + رفع `BUILD_ID` |

### 3) ما لم يتغيّر

- **الديسكتوب**: لم يُمسّ — تجربة الديسكتوب الكاملة محفوظة كما هي.
- **الـ Backend / API**: نفس `useSmartFeed` و `appStore` و `resolveMediaUrl`.
- **الـ Routing**: لا تغيير في المسارات.

## 🚀 كيفية التشغيل

```bash
cd frontend
npm install            # تثبيت الحزم (node_modules غير مضمّن)
npm run dev            # تشغيل بيئة التطوير
npm run build          # بناء الإصدار للإنتاج
```

افتح المتصفح على وضع الموبايل (DevTools → Toggle device toolbar) لمعاينة التصميم الجديد.

## 📱 معايير الاستجابة والوصول

- ✅ **mobile-first**: التصميم الأساسي للأجهزة الصغيرة.
- ✅ **safe-area**: دعم كامل لـ notch / home indicator (iOS).
- ✅ **touch targets ≥ 44×44**: مطابق لـ Apple HIG و WCAG.
- ✅ **RTL**: واجهة عربية كاملة (الاتجاه من اليمين إلى اليسار).
- ✅ **aria labels**: على كل الأزرار والروابط.
- ✅ **focus-visible**: حلقة تركيز بنفسجية لمستخدمي لوحة المفاتيح.
- ✅ **prefers-reduced-motion**: متوافق (لا توجد رسوم متحركة قسرية).

# 🎬 نظام الستوري — تحديث v59.10

## ملخص الإصلاحات

تم فحص نظام الستوري بالكامل وإصلاح النواقص والمشاكل لضمان دورة كاملة:
**رفع → تحسين → نشر → عرض → حفظ → تفاعل** لكل من المشتركين والمتابعين والأصدقاء.

## 🔧 ما تم إصلاحه في الـ Backend

### `backend/app/core/story_store.py`

| المشكلة | الحل |
|---------|------|
| استخدام `datetime.utcnow()` (مهجور في Python 3.12+) | استبداله بـ `datetime.now(timezone.utc)` |
| تسجيل المالك كمشاهد لقصته بالخطأ | تمييز المالك في `mark_seen` و `add_reaction` |
| **لا يوجد سجل مفصّل للمشاهدات** | إضافة `StoryView` يحفظ من شاهد + متى |
| **لا يوجد API لقائمة المشاهدين** | إضافة `get_viewers()` للمالك فقط |
| حذف القصة لا يحذف الملف الفعلي (تسرب تخزين) | إضافة `_try_delete_media_file()` آمن |
| القصص المنتهية تترك ملفاتها على القرص | الحذف التلقائي عند `_purge_expired()` |
| حد الرسم 5000 حرف فقط (يضيع الرسومات) | رفعه إلى 200KB |
| لا يوجد دعم استطلاع تصويتي حقيقي | إضافة `vote_poll()` مع منع التكرار |
| لا عنوان مخصص للقصص المميزة | إضافة `highlight_title` و `set_highlight_title()` |
| كتابة JSON غير ذرّية (قد تتلف عند الإنهاء) | كتابة `.tmp` ثم `replace()` |
| لا يوجد hook للإشعارات | `set_mention_hook()` و `set_new_story_hook()` |
| `auto_delete_hours` غير محدود (يمكن إساءة الاستخدام) | تحديده بين 1 و 72 ساعة |

### `backend/app/api/routes/stories.py`

✅ مسارات API جديدة:
- `GET  /stories/{id}/viewers` — قائمة من شاهد قصتي (للمالك)
- `POST /stories/{id}/poll/vote` — تصويت على استطلاع
- `POST /stories/{id}/highlight/title` — تسمية لقطة مميزة
- `POST /stories/{id}/highlight` يقبل الآن `{ title }` اختياري

✅ تحسينات:
- `view_story` يمرر الآن `viewer_user_id` (المالك لن يُحسب)
- `react_story` يمرر `viewer_user_id`
- `reply_story` يطلق إشعار `story:reply` لصاحب القصة
- `add_story` يطلق `story:new` و `story:mention` (عبر hooks)
- `_load_friend_ids` محمي بـ try/except (لا يكسر إذا تعطلت قاعدة البيانات)

### `frontend/src/components/stories/StoriesBar.jsx`

| المشكلة | الحل |
|---------|------|

### `frontend/src/components/stories/StoryEditor.jsx`

| المشكلة | الحل |
|---------|------|
| `drawing_data` يُرسل فقط إذا < 4.5KB (يضيع الرسومات) | رفع الحد إلى 120KB |
| إغلاق المحرر بدون تأكيد يفقد العمل | تأكيد عبر `confirm()` عند الخروج مع تعديلات |

### `frontend/src/components/stories/StoryViewerEnhanced.jsx`

🆕 **ميزات جديدة كبرى:**

1. **قائمة المشاهدين (Bottom Sheet)**
   - المالك ينقر على عدّاد المشاهدات → يفتح bottom sheet
   - يعرض صورة + اسم + وقت المشاهدة لكل شخص

2. **التصويت داخل الستوري**
   - يعرض الاستطلاع مع نسب التصويت المباشرة
   - منع التصويت مرتين لنفس الخيار
   - تحديث متفائل (optimistic update)

3. **التحكم بالصوت** 🔊/🔇
   - زر كتم/تشغيل للفيديو (مفقود سابقاً)

4. **تنزيل القصة في الجهاز** ⬇️
   - زر تنزيل يحفظ الصورة/الفيديو محلياً

5. **إبراز (Highlight) من العارض** ⭐
   - زر مباشر لإضافة/إزالة من القصص المميزة
   - مع عنوان مخصص

6. **Long-press للإيقاف** بدل mousedown العادي
   - 180ms threshold يفرّق بين ضغطة عابرة وضغط طويل

7. **Toast notifications** بدل alert

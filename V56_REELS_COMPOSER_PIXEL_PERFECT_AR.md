# تقرير الإصدار v56 — صفحة رفع/إنشاء الريلز (Pixel-Perfect)

## ✅ المُنجَز

تم استبدال صفحة رفع الريلز القديمة بالكامل، وأصبح النموذج الجديد هو **المعتمد** لكل مسارات الريلز.

### 1) صفحة جديدة: `src/pages/ReelComposer.jsx`
صُمِّمت بمطابقة **1:1** للصورة المرجعية:

- **الشريط العلوي**: زر إغلاق (X) — كبسولة "إضافة صوت 🎵" — زر الإعدادات (⚙).
- **عمود يمين (الكاميرا/الصوت)** — 7 أزرار:
  1. **قلب** (تبديل بين الكاميرا الأمامية/الخلفية).
  2. **الفلاش** (إيقاف / تشغيل / تلقائي).
  3. **الجودة 1080p** (مع 480p/720p/2K/4K).
  4. **الميكروفون** (تشغيل/إيقاف).
  5. **فلاتر الضوضاء** (تشغيل/إيقاف).
  6. **كتم الأصوات**.
  7. **الترجمة** التلقائية للمحتوى.
- **عمود يسار (تأثيرات الفيديو)** — 8 أزرار:
  1. **المدة 15s** (مع 30/60/90).
  2. **السرعة 1x** (0.3x .. 3x).
  3. **تحسين** بصري.
  4. **الفلاتر** (warm/cool/vintage/mono/enhance).
  5. **المؤثرات** (sparkle/glow/shake/zoom).
  6. **المؤقت** (3s/5s/10s قبل التسجيل).
  7. **التخطيط 9:16** (1:1/4:5/16:9).
  8. **تجميل** الوجه.
- **الوسط السفلي**: زر تسجيل دائري بنفسجي كبير مع زر إلغاء (×) وزر تأكيد ونشر (✓).
- **تابات النوع**: قوالب · صورة · **ريلز (نشط)** · لايف · نشر.
- **الشريط السفلي**: **المعرض** (يفتح ملفات الجهاز) · **المسودات**.
- **شريط تقدّم رفع** يظهر فوق الأزرار أثناء النشر مع نسبة مئوية حقيقية.

### 2) ربط الـ Routing
- مسار جديد معتمد: `/reels/new`
- مسارات بديلة (للتوافق الخلفي): `/reels/upload` و `/reels/composer`
- تم تعديل `src/App.jsx` لإضافة الـ lazy import للصفحة الجديدة.

### 3) تعطيل صفحة الرفع القديمة
- في `src/components/mobile/BottomNav.jsx` كان زر "ريل جديد" يُطلق حدثاً `yamshat:open-reel-composer` لا يستمع له شيء (مكسور).
- الآن يقوم بـ `navigate('/reels/new')` مباشرة إلى الصفحة المعتمدة الجديدة.
- لم يُحذف الملف القديم `VideoUploader.jsx` لكي تبقى الخدمات الفرعية متاحة لأي صفحات إعدادات أخرى، لكنه لم يعد المستخدم عند الضغط على زر "رفع ريلز" في الأسفل.

### 4) دعم الرفع الحقيقي
- التسجيل المحلي عبر `MediaRecorder` (WebM/Opus).
- اختيار فيديو من المعرض (MP4/WebM/MOV/M4V/MKV/3GP) بحدّ أقصى 500MB.
- الرفع عبر `mediaUploadService.uploadFile()` مع `purpose: 'reel-upload'`.
- إنشاء سجل الريل عبر `POST /reels` ثم fallback إلى `POST /reels/create`.
- معاينة فلاتر CSS مباشرة قبل التسجيل (enhance/warm/cool/vintage/mono + beautify).

### 5) لوحات الإعدادات (Bottom Sheets)
- لوحة لكل من: المدة / السرعة / الجودة / التخطيط / الفلاش / الفلاتر / المؤثرات / المؤقت / الصوت.
- لوحة "إعدادات الريل" الموحّدة من زر التروس العلوي، مع زر سريع للانتقال إلى `/settings/reels`.

## 🚀 كيف تستعمله
1. افتح التطبيق → اذهب إلى تبويب الريلز.
2. اضغط الزر الأوسط في الشريط السفلي (زر الإنشاء +).
3. ستفتح الصفحة الجديدة المطابقة 100% للصورة.
4. سجِّل أو اختر من المعرض، اضبط الإعدادات، ثم اضغط ✓ للنشر.

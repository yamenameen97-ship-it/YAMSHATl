# Render Deployment Configuration for Yamshat Frontend

## Build Configuration
- Node.js Version: 20.x or 22.x
- npm Version: >= 10
- Build Command: `npm run build`
- Start Command: `npm run preview` or serve via Nginx

## Environment Variables
No environment variables are required for the frontend build.

## Notes
- The application is a React SPA using Vite
- PWA support is enabled
- Service Worker is configured
- Nginx is used to serve the built application
